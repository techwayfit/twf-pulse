🟦 Operator Role
Can view users, sessions, and audit logs. Can update display names, disable accounts, extend session expiry, and lock sessions.

Cannot: delete accounts, change emails, delete sessions, or edit activity configs.
🟣 SuperAdmin Role
Full access. All Operator permissions plus: update emails, delete accounts, delete sessions, edit activity Config JSON, and manage operator accounts.

All actions are audited.
