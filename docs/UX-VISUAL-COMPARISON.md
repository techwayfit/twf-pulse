## Key Visual Improvements

### Typography
- **Scale:** Larger headings, better hierarchy
- **Spacing:** More breathing room between elements
- **Weight:** Strategic use of bold for emphasis

### Color Usage
- **Gradients:** Blue to mint for premium feel
- **Tints:** Subtle background colors for activity types
- **Borders:** Colored left borders for visual interest

### Depth & Shadows
- **Layered shadows:** Multiple shadows for realistic depth
- **Hover states:** Cards lift on hover (8px up)
- **Focus states:** Blue glow for better accessibility

### Animation
- **Subtle:** Smooth transitions (0.3-0.4s)
- **Purposeful:** Shine effects, pulses, slides
- **Accessible:** Respects reduced-motion preference

### Spacing
- **Consistent:** Using CSS variables
- **Generous:** More whitespace between sections
- **Responsive:** Adjusts for mobile devices
